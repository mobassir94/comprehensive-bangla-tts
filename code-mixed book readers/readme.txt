this directory contains code that were used to generate various code-mixed books (tafsir and hadiths mainly)

datasets :
1. all books from ihadis.com are scraped and published CSV versions here -> https://www.kaggle.com/datasets/nazmuddhohaansary/i-hadith-all-books

2. tafsir bayan and tafsir zakaria comes from this dataset -> https://www.kaggle.com/datasets/mobassir/bangla-quran-with-tafsir

3. tafsir-al-jalalayn comes from this source https://www.kaggle.com/datasets/alizahidraja/quran-english which then later gets translated into bangla in this directory : https://github.com/mobassir94/comprehensive-bangla-tts/tree/main/prepare_dataset

4. tafsir ibn kathir was scrapped and made available here : https://github.com/mnansary/hadith-srcapper/tree/main/alimorg which we later translated back to bangla-arabic format in this directory : https://github.com/mobassir94/comprehensive-bangla-tts/tree/main/prepare_dataset
